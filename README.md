# Thupten Wangpo

I build developer tools, retrieval systems, and applied ML products for teams that need software to be useful, inspectable, and fast to ship.

I am strongest where product judgment, systems thinking, and hands-on implementation meet: turning ambiguous workflows into reliable tools, making messy data usable, and building demos that show how a system behaves in the real world.

If you are a recruiter, hiring manager, teammate, or potential collaborator, this page is meant to help you quickly see where I can help and which projects are worth opening first.

## Where I Can Help

- Developer tools and workflow automation that reduce manual coordination.
- Retrieval and search systems over documents, code, research, and operational context.
- Applied ML demos that connect data prep, modeling, evaluation, and user-facing output.
- Local-first software where privacy, auditability, and operational control matter.
- Technical writing that makes a system, tradeoff, or decision easy for other people to inspect.
